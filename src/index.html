Travel Guide

Travel Guide is a full stack application developed with Node, and ExpressJS. The focus of the project was to explore database design and implementation. Using modelling and diagraming, we designed and normalized the relational database through a series of iterations. The database is accessed through a web interface, and RESTful API. We used Docker in the project which allowed for ease and consistency in both development and deployment.

Gambyt Solutions

Gambyt Solutions is a distributed task tracking app. The project was built to explore distributed systems. The backend is built in Java using Spring Boot. A proxy server manages all requests, and connections to the distrubuted backend. In order to ensure that the service is always available, each component is monitored and will automatically restart in the event of a failure.
